## 🌐免费域名类型及申请途径
1. 顶级域名（直接注册）
    - Freenom（已停止新注册）
提供 .tk、.ml、.cf、.gq、.ga 等后缀，曾是唯一免费顶级域名服务商，但目前因滥用问题暂停新注册 。
   - 替代方案：暂无同类顶级域名免费服务，建议转向二级域名。


1. 二级域名（主流选择）  
   
   |服务商|域名后缀示例|特点|托管支持|

## 🖥是否需要购买服务器
>*不需要！ 免费域名可通过以下方式直接使用，无需自备服务器.* <br>
>*&#x26a0;免费毕竟是免费，也有很大风险，可以用于个人测试和学习，不建议用于正式项目，有能力建议购买正规域名。*
1. 托管至免费平台:
   -  GitHub Pages：绑定域名后，自动托管静态网站（支持 Jekyll 生成页面） 。
   - Cloudflare Pages/Vercel：支持静态站点部署，免费且自带全球 CDN。
 

2. DNS 解析至现有服务-
将域名通过 DNS 指向：
   - GitHub Pages 的默认地址（如 username.github.io）
   - 或已部署服务的 IP/网址（如第三方博客平台）。
 

3. 通过托管服务商获取域名
   - Hostinger/Scala Hosting 等提供 首年免费域名（如 .com），但需购买其托管套餐。

## 🚀操作流程-以 Gname + Cloudflare 为例

1. 注册域名 yourname.eu.cc
   - 访问 [Gname](https://gname.vip)，利用邮箱或手机号注册登录账号，回到首页领取免费注册券，完成验证。
![ym1](https://i.postimg.cc/QxdzL6GN/ym1.webp)
   - 领取免费注册券后，点击`免费注册`，`搜索栏`填入自己想要的域名前缀（如 yqxmm），选择 `.eu.cc `<mark>(后缀一定是.eu.cc,否则无法用到免费优惠券)，</mark>点击`立即注册`。
   - 查看后方金额是否为“0.00”。
     - <a id="one">为“0.00”后，勾选`我已阅读并同意《用户协议》`，点击`确认结算`,弹出二级窗口 再次确认，最后点击`立即支付`，等待系统加载完毕，点击左下角`管理我的域名`。</a>
     ![ym2](https://i.postimg.cc/d13NgBxT/ym2.webp)
     ![ym3](https://i.postimg.cc/cHCkPmb3/ym3.webp)
     - [若金额不为“0.00”，请后退查看优惠券有没有领取到或是否选择了<mark>.eu.cc </mark>后缀的域名](#one)。
   - 进入如下界面查看自己的域名信息是否正确。
     ![ym4](https://i.postimg.cc/DZ0pRgMs/ym4.webp)

1. 托管至 Cloudflare
   - 访问[Cloudflare](httpshttps://dash.cloudflare.com/) ,注册并登录账号，点击`添加域名`，填写刚才注册好的域名，其他内容保存默认，点击`Continue`。
   ![ym5](https://i.postimg.cc/9M0gspnw/ym5.webp)
   ![ym6](https://i.postimg.cc/bJ5VPt7n/ym6.webp) 
   - 选择 `Free` 套餐，点击 `Continue`。
   ![ym7](https://i.postimg.cc/ZRQsmdG3/ym7.webp)
   - 直接至页面下方点击 `继续前往激活`，复制系统生成的 DNS 记录。
   ![ym8](https://i.postimg.cc/xCZpQzwK/ym8.webp)
   ![ym9](https://i.postimg.cc/k4HYCb0N/ym9.webp)
   - 回到[Gname](https://gname.vip)，进入控制面板页面，选择刚才注册的域名，点击后方`管理`，下方`常用功能`找到`修改DNS`,将复制的 DNS 记录粘贴，保存后等待系统加载，刷新后查看是否生效。
   ![ym10](https://i.postimg.cc/50rKJChn/ym10.webpT)
   ![ym11](https://i.postimg.cc/cHzkWnp9/ym11.webp)
   - 生效后回到[Cloudflare](httpshttps://dash.cloudflare.com/)，点击`我已更新名称服务器`,跳转后点击`立即检查名称服务器`，此时需要进行必要的系统加载等待（可以等待1-2min刷新网页查看进度，出现以下界面表示DNS切换成功）。
   ![ym12](https://i.postimg.cc/k4HYCb0h/ym12.webp)
   ![ym13](https://i.postimg.cc/d1fNFCMP/ym13.webp)
2. 配置 DNS 记录（指向服务器/服务）-
   *让你的域名（yqxmm.eu.cc）解析到你的 cloundflare 服务器或其他托管平台。*

- 进入 Cloudflare 左侧菜单 DNS → DNS 记录。
- 添加 A 记录（用于指向服务器 IP）：

   |类型	|名称	|IPv4 地址	|代理状态|
   |------|------|------|------|
   |A	|@	|192.138.x.x|橙色云朵|

   ![ym14](https://i.postimg.cc/d3THV0F5/ym14.webp)
   ![ym15](https://i.postimg.cc/Gts7mpRV/ym15.webp)
**说明：**

- `@ `代表根域名 yqxmm.eu.cc；
- 若需子域名（如 pan.yqxmm.eu.cc），名称填 pan；
- 橙色云朵：开启 Cloudflare CDN 和防护（推荐）；
- 灰色云朵：仅 DNS 解析（直连服务器）；

<summary>💉什么是ipv4/ipv6地址？</summary>
  
  - 区分公网 IP 类型：

    |IP|地址格式|备注|

  - 如何获取 IPv4/IPv6 地址？

    ```
    # 优先使用 curl（简单可靠）
    curl -4 ifconfig.co       # 获取 IPv4 地址

    ```  
    - 🌐 浏览器访问检测网站（无需登录服务器）-访问以下任意网站自动显示公网 IP：
    - https://ifconfig.co
    - https://whatismyip.com
    - https://ipinfo.io
    - https://checkip.amazonaws.com（极简页面）

</details>

4. 配置 SSL/TLS（启用 HTTPS）
**目标让网站通过 https://yqxmm.eu.cc 安全访问。**


   - 进入左侧菜单 SSL/TLS → 概述。
   - 选择 “Full (strict)” 模式（强制 HTTPS，最安全）。
   ![ym16](https://i.postimg.cc/tJVS4gbf/ym16.webp)
   - 开启 “始终使用 HTTPS”（在 SSL/TLS → Edge Certificates 中设置）。
   ![ym17](https://i.postimg.cc/Hn7ZkLmF/ym17.webp)
   ✅ 效果：几分钟后 Cloudflare 会自动签发免费 SSL 证书，访问时显示小绿锁。
1. 完成托管
   - 回到[Cloudflare](httpshttps://dash.cloudflare.com/)个人界面，查看个人域名是否激活成功。

## ⚠️ 注意事项
1.免费域名的局限性
- 部分服务可能回收域名（如 Freenom 曾因活跃度不足回收） 。
- 后缀非主流（如 .eu.cc），不适合商业正式用途。
 

2.国内用户特殊问题
- 国内注册 .cn 等域名必须备案，且备案需租用3个月以上国内服务器（约200元成本） 。
- 国外注册域名（如 NameSilo 购买 .com）无需备案，但国内访问可能较慢 。
 

3.长期使用建议
- 重要项目推荐购买正规域名（如 .com，约 $10/年），避免免费服务的不稳定性。
- 定期检查域名是否过期，及时续费。
