# HUAWEI ohos 侧载
## 🎉准备工作
>  - 以[Kazumi hap](https://github.com/ErBWs/Kazumi/releases/tag/1.9.4) 安装包为例
> - 一台<mark>任意非鸿蒙 5.0 及以上</mark>的设备。
---
## 🎡小白调试助手
### *1.下载安装*
1. 从 GitHub 仓库 [Release](https://github.com/likuai2010/auto-installer/releases) 中下载两个平台的安装包，其中一个需要为 .hap 安装包，用于后续软件更新使用；
2. 另一个可以为任意平台(windows/macos/linux)。

 &#x2139;&#xfe0f;**注意事项**

>Windows: 首次打开会提示安装 Java，下载安装即可。<br>
>macOS: 首次运行需要在终端运行` xattr -d com.apple.quarantine /Applications/`小白调试助手.app。需要 `Java 环境`。<br>
>Linux: 不能使用 root 权限，依赖 zenity。

### *2.开启无线调试*
1.打开`设置` - `设备名称`，进入`关于本机`页面。快速点击5下`软件版本`打开`开发者模式`。

2.转到`设置` - `系统` - `开发者选项`页面，找到并点击`无线调试`，在二级页面中开启`无线调试`，获取设备的 IP 地址和端口。
<img src="https://kazumi.app/assets/wireless.BPjalEmW.png" width="50%">

### *3.开发者账号*
1.侧载中需要使用开发者账号进行签名。开发者账号未实名时侧载后的应用仅有 14 天有效期，实名后会延长至 180 天，因此首先需要进行实名认证。

2.打开<u>[华为开发者联盟](https://developer.huawei.com/consumer/cn/console/setting/profile)</u> - 开发者信息页面-注册或登录华为账号。
填写个人基本信息进行实名认证。
### *4.侧载 Kazumi*
1.登录账号
点击登录账号，在浏览器中登录`华为账号`，点击`允许访问`，等待出现`登录成功`！请返回！。

2.连接设备
点击连接设备，在弹出窗口中输入上面获取的 IP 地址和端口，点击按钮进行连接。首次连接需要`信任该设备`，选择`信任`.
<img src="https://kazumi.app/assets/device.BORET4xz.png" width="50%">
3.选择 .hap 文件
点击选择，选择下载好的 `Kazumi hap` 包，完成后会在软件内显示软件包名及支持安装的设备。
<img src="https://kazumi.app/assets/ready.D-TvZCf3.png" width="50%">
4.侧载安装
点击开始调试，等待全部操作成功完成侧载。若出现错误可根据错误信息进行操作解决，并重新开始调试。

### *5.便捷更新*
使用相同方法侧载小白调试助手的 hap 包，后续软件内更新时会下载 hap 包并发送到小白调试助手，只需要登录华为账号并连接设备就可以便捷侧载。
