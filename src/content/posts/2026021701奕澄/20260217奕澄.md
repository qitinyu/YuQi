# Markdown

### 引言

- Markdown是一种‌轻量级标记语言，由 `约翰·格鲁伯（John Gruber）`于2004年创建，允许用户使用易读易写的纯文本格式编写文档，并自动转换为HTML、PDF等结构化格式。

### 特点


- 轻量化、易读易写。
- 支持图片、图表、数学式，网站泛用性高。
- 应用广。在 GitHub、Reddit、Diaspora、Stack Exchange、OpenStreetMap、SourceForge、简书等。

### 优势

- 世界上最流行的博客平台WordPress和大型CMS如Joomla、Drupal都能很好的支持Markdown。完全采用Markdown编辑器的博客平台有Ghost和Typecho等。
- Markdown可以快速转化为演讲PPT、Word产品文档甚至是用非常少量的代码完成最小可用原型。
- 通过OCR技术（如Mistral OCR3）自动从文档中提取文本和嵌入式图像并输出为Markdown格式，帮助下游系统理解文档结构，适用于企业级流程如发票处理和文档数字化。

### 格式

- README.md

# 奕澄Markdown

### 介绍

- [开发者bilibili主页](https://space.bilibili.com/330615268)
- [演示视频](https://www.bilibili.com/video/BV1b8bmzuEyd)
- [下载直链](https://appgallery.huawei.com/app/detail?id=com.lotoze.md&channelId=SHARE&source=appshare)

### 特点

1. 多模式
   - HarmonyOS NEXT最简洁实用的markdown工具。
   - 所见即所得模式
   - 即时渲染模式
   - 分屏预览模式

2. 编辑

   - 渲染大纲
   - 编辑退出自动保存
   - 独立预览

3. 导出

   - 打印，可连接打印机
   - pdf导出
4. 浏览

   - markdown文件列表
   - pdf文件列表
   - 列表操作
   - 文件编辑（PDF只支持预览）
   - 预览
   - 同鸿蒙左滑便捷操作
   - 鸿蒙设备同账号同步
