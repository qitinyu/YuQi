# 什么是OpenList？
### 来源：
- 众所<del>不</del>周知：Alist 已经确认被“卖”，版本3.4之后存在风险。即使你使用的是之前的版本，当你的网盘token失效，需要更新的时候，走的还是 Alist 的服务器，依然存在网盘泄密的风险。当然人都有自己的选择，这里不过多评价。
- 目前社区出现的 fork 当中，替代品 OpenList 已经上线，本文简单介绍一下 OpenList的部署安装和设置过程。而OpenList是一款‌开源的、基于AList项目分支开发的文件列表程序‌，常被用作‌多网盘聚合管理工具‌。 ‌
### 简述：
**‌核心功能‌：**
- 它能将用户分散在‌本地存储、阿里云盘、百度网盘、夸克网盘、OneDrive、WebDAV等数十种存储服务‌中的文件，统一整合到一个美观的Web界面中进行管理。 ‌

**‌主要特点‌：**
- ‌一站式管理‌：用户无需在不同应用间切换，即可实现文件的浏览、在线播放（支持4K视频）、上传下载等操作。 ‌

**‌开源与可部署‌：**
- 作为AList的社区维护分支，它强调可持续性和安全性，支持通过Docker、二进制文件等多种方式在NAS（如极空间、飞牛NAS）、服务器或个人设备上部署。 ‌

**‌高级功能‌：**
- 除基础聚合外，还提供文件预览、离线下载、权限密码保护、WebDAV/FTP服务开启等企业级功能。

# *配置*
### 1、Openlist/openlist-desktop安装包下载
>**1.1**. 官方文档地址：https://doc.oplist.org/  

>**1.2**. Openlist安装包下载地址：  
OpenList官网：https://github.com/OpenListTeam/OpenList  
   <mark>注意下载_windows_amd64.exe</mark>  

>**1.3**. Openlist-desktop安装包下载地址：
https://github.com/OpenListTeam/OpenList-Desktop/releases
<mark>注意下载Windows：OpenList-Desktop_x.x.x_{arch}-setup.exe

### 2、Openlist/openlist-desktop安装

>- 选择任意盘创建一个文件夹，如：`D:\OpenList`；
>- 将下载好的安装包放到`D:\OpenList`文件夹，分别创建`Openlist`和`Openlist-Desktop`两个文件夹；
>- 下载并解压`Openlist-Desktop_x.x.x_{arch}-setup.exe`到`D:\OpenList-·`文件夹；
>- <mark>右键管理员身份运行（首次安装需要）</mark>，选择`Openlist-Desktop_x.x.x_{arch}-setup.exe`安装程序，根据提示一步步安装即可；
>- 安装完成后，自动打开或双击桌面`Openlist-Desktop.exe`运行程序，
  做如下配置<mark>（先进行配置后进行本地挂载</mark>）：
